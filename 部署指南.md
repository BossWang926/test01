# GitHub 仓库创建和推送指南

## 方法 1：使用 GitHub 网页创建（推荐）

### 步骤 1：在 GitHub 创建新仓库

1. 访问：https://github.com/new
2. 填写仓库信息：
   - **仓库名称**：`my-portfolio-website`
   - **描述**：一个现代简约风格的个人主页，使用 Tailwind CSS 构建
   - **可见性**：选择 Public ☑️
   - **初始化选项**：全部不选（不要添加 README、.gitignore 或 license）
3. 点击 **Create repository**

### 步骤 2：推送代码到 GitHub

创建完成后，GitHub 会显示推送命令。运行以下命令：

```bash
git remote add origin https://github.com/BossWang926/my-portfolio-website.git
git branch -M main
git push -u origin main
```

---

## 方法 2：使用 GitHub CLI（需要重启终端）

安装完成后，重新打开终端，然后运行：

```bash
# 登录 GitHub
gh auth login

# 创建并推送仓库
gh repo create my-portfolio-website --public --description "一个现代简约风格的个人主页，使用 Tailwind CSS 构建" --source=. --push
```

---

## 验证成功

推送成功后，访问：
https://github.com/BossWang926/my-portfolio-website

你将看到：
- index.html 文件
- .gitignore 文件
- 完整的提交历史

---

## 启用 GitHub Pages（可选）

如果要在网上访问你的网站：

1. 访问仓库的 **Settings** 页面
2. 在左侧菜单找到 **Pages**
3. 在 **Build and deployment** 下：
   - **Source**：选择 Deploy from a branch
   - **Branch**：选择 main / root
4. 点击 **Save**

几分钟后，你的网站将在以下地址可用：
https://bosswang926.github.io/my-portfolio-website/
